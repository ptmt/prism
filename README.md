prism
=====

Jenkins Build Status: http://jenkins.phinitive.com/buildStatus/icon?job=PRISM 


For building solution under Linux make sure you have installed mono 3.2+

or use vagrantup:
````
$ vagrant up
$ vagrant ssh
````

then run `build.sh` and start:

````
$ ./build.sh
$ mono build/Prism.App.exe &
````
